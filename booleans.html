1.5 The Arithmetic of Booleans

Exercise 7: Add the following two lines to the definitions area:

(define sunny #true)
(define friday #false)

Now create an expression that computes whether sunny is false or friday is true. So in this particular case, the answer is #false. (Why?)

Racket:

(or (not sunny) friday)

Clojure: 

(def sunny true)
(def friday false)

(or (not sunny) friday)
