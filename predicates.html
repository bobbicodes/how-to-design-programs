1.7 Predicates: Know Thy Data

Exercise 9: Add the following line to the definitions area:

(define in ...)

Then create an expression that converts the value of in to a positive number. For a String, it determines how long the String is; for an Image, it uses the area; for a Number, it decrements the number by 1, unless it is already 0 or negative; for #true it uses 10 and for #false 20.

Racket:

(if (string? in) (string-length in) (if (image? in) (* (image-width in) (image-height in)) (if (number? in) (if (< in 1) in (- in 1)) (if (boolean? in) (if in 10 20) "oops..."))))

Clojure: 

(def in 0)

(if (string? in)
    (count in)
  (if (number? in)
    (if (< in 1)
      in
      (- in 1))
    (if (boolean? in)
      (if in
        10
        20)
      "oops...")))
